# Bug Tracker

Log bugs here before fixing them. One file per bug.

## Naming convention
`YYYY-MM-DD-short-description.md`

## Template
```
## Bug: <title>
**Reported:** YYYY-MM-DD
**Status:** open | in-progress | fixed

### Reproduction
1. Step one
2. Step two

### Root Cause
(fill in when known)

### Fix
(fill in when resolved)
```
